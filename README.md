# SportSee

## Description

SportSee est une application de coaching sportif permettant aux utilisateurs de suivre leurs sessions d'entraînement et les calories brûlées. Cette version du projet se concentre sur la page profil de l'utilisateur, développée avec React et utilisant des graphiques pour visualiser les données.

## Installation

### Prérequis

- Node.js 
- npm ou yarn

### Étapes

1. Clonez le dépôt
   git clone [https://github.com/votre-utilisateur/sportsee.git](https://github.com/yazid78/Sport_See.git)

### Démarrer le Backend

1. Naviguez vers le répertoire `Back_End` :
   cd Back_End
2. Lancez le serveur : npm start
    
### Démarrer le Frontend

1. Naviguez vers le répertoire `Front_End` :
   cd Front_End
2. Lancez le serveur : npm run dev
